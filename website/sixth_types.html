Задание 6 - Программа с условным оператором
Виды задания №6
1. Задание с двумя переменными
В этом типе задания дается две переменные, которые сравниваются либо между собой, либо с определенными числами.
В зависимости от результата этого сравнения программа выводит один из заранее заданных результатов, например, "ДА" или "НЕТ".
Задание заключается в том, чтобы посчитать, сколько раз программа выведет один из этих результатов, используя несколько различных пар значений переменных.
2. Задание с двумя переменными и параметром
В этом типе задания даются две переменные и параметр, который также влияет на результат проверки.
Программа выполняет сравнение двух переменных между собой или с параметром, и в зависимости от этого выводит один из заранее определенных результатов (например, "YES" или "NO"). Задание заключается в нахождении такого значения параметра, при котором программа будет выводить один из этих результатов заданное колличество раз.
